> **Note :** Dépose tes fichiers audio dans `public/sfx/` pour les rendre accessibles.
> Format recommandé : `.wav` pour la qualité, `.mp3` pour le web.

## Conventions de nommage

```
sfx_[catégorie]_[description]_[variante].wav

Exemples :
  sfx_moteur_pulsar_court.wav
  sfx_arme_tir_energie_v2.wav
  sfx_ambiance_espace_loop.wav
```
