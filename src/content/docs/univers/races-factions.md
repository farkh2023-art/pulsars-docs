## Helix

- **Type** : Corporation technocratique
- **Territoire** : Systèmes du noyau galactique
- **Philosophie** : Contrôle des pulsars = contrôle de l'énergie = pouvoir absolu
- **Signes distinctifs** : Architecture angulaire, uniformes blancs, IA intégrée

## Nomades du Vide

- **Type** : Faction décentralisée
- **Territoire** : Zones interstellaires non revendiquées
- **Philosophie** : Liberté de circulation, refus des frontières énergétiques
- **Signes distinctifs** : Vaisseaux patchwork, culture orale, forte cohésion communautaire

## Conseil de Nova

- **Type** : Gouvernance diplomatique
- **Territoire** : Station Nova — point de neutralité
- **Philosophie** : Équilibre entre les factions, monopole sur les négociations
- **Signes distinctifs** : Symbolisme circulaire, tenues bleu-nuit, protocole strict
